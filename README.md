# Scholarship Finder

A Scholarship Finder interface that allows users to browse scholarships using filtering, sorting, and Local Storage.

## Features

- Scholarship Cards UI
- Filter by Country, Stream, Level
- Sort by Amount and Deadline
- Load More functionality
- Data stored in Local Storage
- Responsive layout

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
